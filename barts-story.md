# Getting Data into Excel Online

Module 1 is using the following datasets:

- 0923 BSc Student allocations (LSBU-Barts Health)
- student allocations

to explain how the first consideration we must have with data is where it's arrived from and the different ways that data is structured. 

It's framed around the question: *before I try to gain any insights from this data, how can I ensure the data is consistent and correct?*
